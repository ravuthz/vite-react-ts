# React typescript project using Vite, ViTest, Eslint, Prettier

I just create a vite project boilerplate for easy to scaffold web application with react using typescript on Microsof Visualt Studio Code.

## To use this template just clone it or using degit

```bash
npx degit ravuthz/vite-react-ts web-project-title
```

## Or just clone it manually
```bash
git clone https://github.com/ravuthz/vite-react-ts web-project-title
```
